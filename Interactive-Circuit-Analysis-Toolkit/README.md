# RC Filter Analysis Toolkit

An interactive Python notebook for analyzing and designing first-order RC low-pass filters.

This project demonstrates how Python can be used to solve practical electrical engineering problems through numerical computation, scientific visualization, and interactive design tools.

## Features

- Calculate RC cutoff frequency
- Plot frequency response
- Interactive component selection
- Interactive filter design
- Engineering verification
- Scientific documentation

## Technologies

- Python
- NumPy
- Matplotlib
- Jupyter Notebook
- ipywidgets

## Engineering Concepts

This project demonstrates:

- RC low-pass filters
- Transfer functions
- Cutoff frequency
- Time constants
- Frequency response
- Engineering design equations

## Example Equations

Cutoff Frequency

$$
f_c=\frac{1}{2\pi RC}
$$

Magnitude Response

$$
|H(f)|=\frac{1}{\sqrt{1+(2\pi fRC)^2}}
$$

Design Equation

$$
R=\frac{1}{2\pi C f_c}
$$
